# Dấu ?. (Optional Chaining)
Dấu này dùng để truy cập an toàn vào các thuộc tính lồng nhau của một Object mà không sợ bị "văng" lỗi nếu Object cha bị null hoặc undefined.

Nếu không có dấu ?.: Giả sử người dùng vào thẳng trang Login (không qua điều hướng từ trang khác), location.state sẽ là null. Khi bạn gọi location.state.from, trình duyệt sẽ báo lỗi ngay: TypeError: Cannot read property 'from' of null.

Khi có dấu ?.: Nếu location.state là null, nó sẽ dừng lại và trả về undefined thay vì báo lỗi.
## code
```bash
import { Navigate, useLocation } from "react-router";
const location = useLocation();
const from = location.state?.from || '/';

return navigate(from, { replace: true });
```

## Ví dụ trong file ProtectedRoute.jsx của bạn:
```bash
import { Navigate, useLocation } from "react-router";

const ProtectedRoute = ({ children }) => {
    const token = localStorage.getItem('laravel_token');
    const location = useLocation();

    if (!token) {
        // Gửi thông tin trang hiện tại qua prop 'state'
        return <Navigate to="/login" state={{ from: location.pathname }} replace />;
    }

    return children;
};
```
