## Tắt trong cấu hình Snippets
Mở Settings trong VS Code (Ctrl + ,).

Tìm kiếm từ khóa: react snippets config.

Tìm mục: "React Snippets › Settings: Import React On Top".

Bỏ tích (Uncheck) ô này. Sau đó, khi bạn gõ lệnh tắt, nó sẽ không tự chèn dòng import React nữa.
